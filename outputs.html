Outputs

Running caldip on a cast produces two types of output: an interactive plot and statistics CSV files.

Interactive plot

Generated by caldip_plot_all.py, the plot is a self-contained HTML file that can be opened in any browser — no internet connection required. This makes it suitable for use at sea.

python caldip_plot_all.py data/proc_calib/msm142_2026/cal_dip/castM4/castM4.caldip.yaml --ctd-sensor 2

The plot shows time series of temperature, pressure, and conductivity for all instruments alongside the CTD reference, with bottle stop periods highlighted. Panels are synchronised — zooming one panel zooms all.

Open example plot (castM4, MSM142)

Statistics CSVs

Generated by caldip_check_all.py:

python caldip_check_all.py data/proc_calib/msm142_2026/cal_dip/castM4/castM4.caldip.yaml --ctd-sensor 2

This produces two CSV files per cast.

Summary statistics ({cast}_summary_statistics.csv)

One row per instrument, taken from the deepest bottle stop of the cast. (This is a single-stop snapshot, not an average across all stops — use the detailed CSV to see all stops.)

Column

Description

serial

Instrument serial number

instrument_type

Instrument family (rbr, sbe37, etc.)

label

Human-readable model label

n_samples

Number of instrument data points in the 2-minute comparison window

bl_press

Pressure of the deepest bottle stop (dbar)

temp_diff_mean

Temperature offset vs CTD at the deepest stop (°C)

temp_diff_std

Standard deviation of temperature within the comparison window (°C)

cond_diff_mean

Conductivity offset vs CTD at the deepest stop (mS/cm)

cond_diff_std

Standard deviation of conductivity within the comparison window (mS/cm)

press_diff_mean

Pressure offset vs CTD at the deepest stop (dbar)

press_diff_std

Standard deviation of pressure within the comparison window (dbar)

RBR thermistors (temperature-only) leave the conductivity and pressure columns as NaN.

Quality flag thresholds

The temp_status, cond_status, and press_status columns compare the instrument offset against these thresholds:

Variable

Default threshold

Flag triggered when

Temperature

±0.005 °C

|instrument − CTD| > 0.005 °C

Conductivity

±0.02 mS/cm

|instrument − CTD| > 0.02 mS/cm

Pressure

±5 dbar

|instrument − CTD| > 5 dbar

These thresholds are appropriate for SBE37 MicroCATs. For RBR thermistors or other instruments with wider specifications, override them in the YAML:

quality_flags:
  temp_threshold: 0.01   # °C
  cond_threshold: 0.05   # mS/cm
  press_threshold: 10.0  # dbar

Output file location

By default, output files are written to the parent of the cast directory (i.e., the cal_dip/ level), not to the YAML directory. The script prints the exact paths when it runs. Use --output-dir to specify a different location:

python caldip_check_all.py castM4/castM4.caldip.yaml --output-dir results/

Batch processing

generate_all_caldip_plots.sh runs both scripts across all casts in a cruise:

bash generate_all_caldip_plots.sh
