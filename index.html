Welcome to caldip’s documentation!

caldip is a Python package for calibration dip analysis of oceanographic instruments. It provides tools for processing, analyzing, and visualizing data from calibration dips performed with CTD profiles and multiple instrument types including MicroCATs and RBR thermistors.

For recommendations or bug reports, please visit https://github.com/eleanorfrajka/caldip/issues/new
